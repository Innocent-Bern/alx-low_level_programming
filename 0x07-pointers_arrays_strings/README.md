# 0x07-pointers_arrays_strings
## 0-memset.c
* _memset : Function that fills memory with a constant byte
* @s: string input
* @b: string input
* @n: integer input
* Return : A pointer to the memory area s
## 1-memcpy.c
* _memcpy : Function that copies memory area
* @dest: string input
* @src: string input
* @n: integer input
* Return : A pointer to the memory area s
